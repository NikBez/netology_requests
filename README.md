# Упражнения для закрепления навыков работы с библиотекой requests

### Порядок локальной работы:

Клонируем репозиторий:
```
git clone <link> 
```
Устанавливаем pipenv, если не установлен:
```
pip3 install pipenv
```
Устанавливаем зависимости:
```
pipenv install
```

### Описание содержимого:
